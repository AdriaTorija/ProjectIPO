# ProjectIPO
Prototype of application for booking table. Year: 2020

## Abstract
This app lets an user see the calendar of the month and see the occupancy of the restaurant, depending on the color: 
  -Green: Empty
  -Yellow: More than 50% occupancy
  -Red: Full
